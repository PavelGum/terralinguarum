    Lesson 06

    Урок№6¶

    Письмо¶

    𓆓 - "змея" обозначет звук [ḏ], читаем как [dʒ] 𓂧 - "рука-кисть" обозначает звук [d] 𓄔 - "ухо животного" обозначает сочетание звуков sḏ [s-dʒ] 𓋴 - детерминатив "сложенная одежда", обозначает звук [s]

    Словарь¶

    𓈖𓈖 - n - предлог "к" 𓆓𓂧 - ḏd - говорить 𓄔𓅓 - sḏm - слушать/слышать

    Этимологическая справка¶

    𓄔𓅓 < 𓋴𓆓𓄔𓅓

    Текст¶

    1. 𓆓𓂧 𓊃𓀀𓏤
    2. 𓄔𓅓 𓊃𓏏𓁐
    3. 𓄔𓅓 𓊃𓏏𓁐 𓈖 𓊃𓀀𓏤
    4. 𓆓𓂧 𓊃𓏏𓁐 𓄔𓅓 𓊃𓀀𓏤 𓈖 𓊃𓏏𓁐
    5. 𓆓𓂧 𓊃𓀀 𓇋𓅓 𓂋𓂝𓇳 𓅓 𓊪𓏏𓇯 𓊃𓏏𓁐 𓅓 𓉐𓏤
    6. 𓆓𓂧 𓊃𓏏𓁐 𓅱𓃀 𓈖𓇳 𓂋𓂝𓇳 𓊃𓀀𓏤 𓅓 𓉐𓏤

    Ответы:¶

    1. ḏd s. 2. sḏm st. 3. sḏm st n s. 4. ḏd st sḏm s n st. 5. ḏd s: ym rʕ m pt, st m pr. 6. ḏd st: wbn rʕ, s m pr.

    Перевод:¶

    в скобках "()" слова, которые не указаны в египетском тексте 1. Мужчина говорит. 2. Женщина слушает. 3. Женщина слушает мужчину. 4. Женщина говорит, мужчина слушает женщину. 5. Мужчина говорит: В небе солнце. Женщина в доме. 6. Говорит женщина: встает солнце, (а) мужчина (находится) в доме
